# Student Manager (Python OOP)

A simple Python project that demonstrates Object-Oriented Programming concepts
by building a student management system.

## Features

- Add student
- Delete student
- Find student
- Sort students by GPA
- Validate GPA

## Concepts Used

- OOP (Object-Oriented Programming)
- Inheritance
- Static methods
- Class methods
- Sorting with lambda
